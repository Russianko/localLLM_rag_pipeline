# sample

## Context

- **Source:** sample.pdf
- **Created:** 2026-03-20 13:08:31

> [!summary] Summary
> Документ является сублицензионным договором между ООО "Айкинит Софт" (Лицензиат) и ООО "СИСТЕМ"Компания" (Сублицензиат). Лицензиат обязуется предоставить Сублицензиату право использования программы для ЭВМ «Сметаплан», гарантируя, что является правоприобретателем простого права на программу. Договор устанавливает порядок использования программы и условия ее предоставления.

## Key Points

- - **Тезис 1**: Лицензиат обязуется предоставить Сублицензиату право использования программы для ЭВМ «Сметаплан» в порядке, предусмотренном Договором.
  
- **Тезис 2**: Сублицензиат обязуется уплатить Лицензиату обусловленное договором вознаграждение.

- **Тезис 3**: Лицензиат гарантирует, что является правообладателем простого (неисключительного) права на программу для ЭВМ «Сметаплан».

- **Тезис 4**: Программа для ЭВМ «Сметаплан» включена в Единый реестр российских программ для ЭВМ с записью №13458 от 18.03.2024 г.

- **Тезис 5**: Сублицензиату предоставляется право использования программы для ЭВМ на условиях простой (неисключительной) лицензии.

- **Тезис 6**: Сублицензиат вправе использовать программу для ЭВМ следующими способами: простое (неисключительное) право на воспроизведение программы для ЭВМ, ограниченное правом инсталляции, копирования и запуска программы для ЭВМ.

- **Тезис 7**: Использование Сублицензиатом программы для ЭВМ допускается на территории Российской Федерации.

- **Тезис 8**: Сублицензиат не вправе заключать Сублицензионные договоры без дополнительного получения письменного одобрения Лицензиата по каждому такому факту.

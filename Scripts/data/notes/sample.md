## Summary

Договором № МАР-03 между ООО "СИСТЕМ"Компания" (сублицензиат) и ООО Айкинит Софт (лицензиат), предоставляется право на использование программы для ЭВМ «Сметаплан» в России. Лицензиат обязуется предоставить сублицензиату права использования программы, а сублицензиат должен уплатить вознаграждение согласно договору. Программа включена в Единый реестр российских программ для ЭВМ и имеет номер 13458. Порядок использования программы оговаривается в Пользовательском соглашении, которое доступно по ссылке на официальном сайте. Лицензиат обязан подготовить спецификацию с указанием прав на использование программ для ЭВМ и размера вознаграждения, которая направляется электронной почтой в сроки, предусмотренные договором. Факт предоставления прав оформляется Универсальным передаточным документом (УПД), который подтверждает право на использование программ для ЭВМ.

## Key Points

1. Лицензиат обязуется предоставить Сублицензиату право использования программы для ЭВМ «Сметаплан» в порядке, предусмотренном Договором.
2. Программа для ЭВМ «Сметаплан» включена в Единый реестр российских программ для ЭВМ Запись в реестре №13458 от 18.03.2024г.
3. Сублицензиату предоставляется право использования программы для ЭВМ на условиях простой (неисключительной) лицензии.

4. Сублицензиат вправе использовать программу для ЭВМ следующими способами: простое (неисключительное) право на воспроизведение программы для ЭВМ, ограниченное правом инсталляции, копирования и запуска программы для ЭВМ.

5. Использование Сублицензиатом программы для ЭВМ допускается на территории Российской Федерации.

6. Сублицензиат не вправе заключать Сублицензионные договоры без дополнительного получения письменного одобрения Лицензиата по каждому такому факту.

7. Лицензиат по заявке Сублицензиата готовит спецификацию и направляет ее в адрес Сублицензиата на адрес его электронной почты, указанный в п. 10. настоящего Договора.
8. Факт предоставления Сублицензиату права на использование программы для ЭВМ оформляется Универсальным передаточным документом (далее – УПД).
9. Права на использование программ для ЭВМ считаются предоставленными Сублицензиату в момент подписания Сторонами УПД.
10. Лицензиат оформляет в двух экземплярах УПД и направляет их Сублицензиату, который обязан подписать полученные экземпляры УПД и вернуть один экземпляр УПД Лицензиату в 5-ти дневный срок с момента получения, либо в указанный срок представить Лицензиату мотивированные и обоснованные возражения против подписания УПД.
